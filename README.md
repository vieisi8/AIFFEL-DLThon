## 프로젝트 기여

### 사전 훈련 모델 선택
  - 데이터셋이 한국말로 되어 있는 관계로 한국어 사전 훈련 모델 선택
    - klue-bert
    - KoELECTRA
  - 각 모델의 F1-score를 비교해 최종적으로 klue-bert 모델 선택
